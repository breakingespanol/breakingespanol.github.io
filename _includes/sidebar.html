We’ve all said we want to learn a second language, Spanish most likely.

This is our family’s journey into becoming more compassionate for Latin culture by learning the language, traveling to different locations, and immersing ourselves in every day Latin life.

In each episode, we talk about recent experiences in becoming immersed in the language and how hard it can be to actually take the time to learn. We both are self-employed entrepreneurs with a small child, so if we can find the time to learn Spanish, you can as well.
